# 一周计划（增肌/健康抗老优先｜外形第二｜08:00 起床｜24:00 入睡｜每天起床先瑜伽）

> 默认起点（可迭代微调）：蛋白 **140 g/天** 固定优先；热量与碳水按训练日/非训练日波动。  
> 你满意格式后，我再按你的体重、训练基础、器械条件把每餐宏量和训练动作替换成“你的版本”。

---

## 周一（训练日｜下肢力量 + 臀腿）

### 每日目标（放最上面）
- 步数：**9,000–11,000 步**（含饭后散步；至少 2 次 15–20 分钟）
- 力量训练：**75 分钟（含热身/放松）**
- 宏量：**2000 kcal｜P 140 g｜C 220 g｜F 60 g**
- 蔬果/纤维：蔬菜 ≥3 份，水果 1–2 份；纤维 **25–35 g**
- 水：**2.0–2.5 L**
- 睡眠：23:30 开始放松流程；**24:00 入睡**
- 疼痛规则：尖锐痛立刻停；关节不适就缩幅/降重量/换动作

### 时间表
**08:00–08:10** 起床 + 喝水 + 状态检查  
**08:10–08:50** 瑜伽 40 分钟（唤醒+活动度，不练到很累）  
**08:50–09:10** 户外光照 + 轻松走路 15–20 分钟  
**09:10–09:40** 早餐：**500 kcal｜P35 C55 F15**  
**10:50–11:00** 走动 5–10 分钟  
**11:30–12:00** 午餐：**600 kcal｜P40 C75 F15**  
**12:00–12:20** 饭后散步 10–15 分钟  
**14:00–14:15** 训练前加餐：**200 kcal｜P15 C25 F5**  
**15:00–16:15** 力量训练（下肢力量 + 臀腿）  
- 深蹲模式 4×4–6  
- 硬拉模式 3–4×5–8  
- 分腿蹲 3×8–10/侧  
- 腘绳肌 3×10–15  
- 小腿 4×8–12  
- 负重走 4 轮  
**16:15–16:30** 训练后补给：**250 kcal｜P25 C35 F0–5**  
**16:30–17:00** 轻松走 10–20 分钟  
**18:00–18:30** 晚餐：**450 kcal｜P25 C30 F20**  
**20:30–21:00** 饭后散步 15–20 分钟（把步数补到目标区间）  
**21:30–21:40** 关节/姿态微训练 10 分钟（肩胛+臀中肌+核心，轻量）  
**23:30–24:00** 睡前加餐（仅蛋白没达标才加）：**200 kcal｜P20 C0–15 F5–8**

# 迭代规则（让这一周越调越准）
- 若训练日表现疲软/重量上不去：优先把训练日前后 **碳水 +20–40g**  
- 若体脂上升太快：先把非训练日 **碳水 -20–30g**（蛋白不动）  
- 若恢复差/睡眠差：先减训练总量（每个动作少 1 组）或加 1 次轻松走路，不要硬加强度  
- 每周只改 1–2 个变量（否则你根本不知道哪个有效）
