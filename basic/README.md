# 宏量营养素新手版（能直接拿来用的“食物换算”+ 关于 220g 碳水的笔记）

> 这是“估算用”的版本：不用天天称重也能八九不离十。  
> 精准就看包装营养表或称重（同一种食物不同品牌/部位差很多）。

---

## 1）蛋白质：100g 蛋白 ≈ 吃多少？

### 超好用速记
- **熟瘦肉/熟鱼 100g ≈ 25g 蛋白**
- **乳清 1勺 ≈ 25g 蛋白**
- **鸡蛋 1个 ≈ 6g 蛋白**
- **希腊酸奶 200g ≈ 20g 蛋白**（看是否高蛋白款）

### 100g 蛋白 ≈（任选组合）
- **熟鸡胸约 400g ≈ 100g 蛋白**
- **熟瘦牛肉约 400g ≈ 100g 蛋白**
- **熟鱼约 450–500g ≈ 100g 蛋白**
- **鸡蛋 6个 ≈ 36g 蛋白**（所以别指望只靠鸡蛋凑满）
- **乳清 2勺 ≈ 50g 蛋白**
- **硬豆腐约 500–650g ≈ 50g 蛋白**（差异很大，看包装）

### 你这周模板常用：140g 蛋白怎么理解？
- 如果分 **4 餐**：每餐目标 **35g 蛋白**
  - **35g 蛋白 ≈ 熟瘦肉/鱼约 140g**
  - 或 **乳清 1勺(25g) + 希腊酸奶 100g(约10g)**

## 2）碳水：220g 碳水 ≈ 吃多少米饭/面/土豆/燕麦？

### 超好用速记（熟重）
- **熟米饭 1碗（约 150g）≈ 40–45g 碳水**
- **熟面条 1碗（约 200g）≈ 50–60g 碳水**
- **土豆 1个中等（约 250g）≈ 40–45g 碳水**
- **燕麦干重 40g ≈ 25g 碳水**
- **香蕉 1根中等 ≈ 25–30g 碳水**
- **吐司 1片 ≈ 12–15g 碳水**

### 220g 碳水/天（训练日）大概相当于
- **熟米饭约 750–850g ≈ 220g 碳水**
- 换成“更现实的混搭”理解：
  - 早餐：燕麦干 60g（≈ 38g）+ 水果 1份（≈ 25g）→ **~60g**
  - 午餐：熟米饭 300g（≈ 85–90g）→ **~90g**
  - 训练前后：香蕉 1根（≈ 25–30g）+ 少量主食/米饼（≈ 20g）→ **~45–50g**
  - 晚餐：熟米饭 80–120g（≈ 25–35g）→ **~30g**
  - 合计接近 **220g**

### “一眼就会”的碳水小目标
- 想要 **50g 碳水**：
  - 熟米饭 **~180g**（比一碗略多）
  - 或 土豆 **1个中等**
  - 或 熟面条 **~170–200g**
  - 或 燕麦干 **80g**

## 3）脂肪：60g 脂肪 ≈ 吃多少油/坚果/牛油果？

### 超好用速记
- **食用油 1汤匙（约 15ml）≈ 14g 脂肪**
- **坚果 30g（一小把）≈ 15g 脂肪**
- **牛油果 1个中等 ≈ 20–25g 脂肪**
- **花生酱 1汤匙（约 16g）≈ 8g 脂肪**
- **全蛋 1个 ≈ 5g 脂肪**
- **奶酪 30g ≈ 9–10g 脂肪**（看种类）

### 60g 脂肪/天大概相当于
- 油 **2汤匙**（≈ 28g）  
- 坚果 **30g**（≈ 15g）  
- 鸡蛋 **2个**（≈ 10g）  
- 再加肉/鱼自带脂肪 **~7–10g**  
= 约 **60g**

## 4）外食/不称重也能用：手掌估算法
- **蛋白（熟肉/鱼）**：1个手掌大小厚度 ≈ **25–30g 蛋白**
- **碳水（熟饭/面/土豆）**：1捧手的量 ≈ **25–35g 碳水**
- **脂肪（油/坚果）**：1个大拇指的量 ≈ **10–12g 脂肪**

## 5）关于“54 岁能不能吃 220g 碳水”的笔记

### 结论
**能。** 54 岁不是碳水禁令。  
关键看：**训练、步数、总热量**，以及碳水放在训练前后、来源干净。

### 先把数字算清楚（别凭感觉吵）
示例训练日宏量：**2000 kcal / P140 / C220 / F60**
- 蛋白：140×4 = **560 kcal**
- 碳水：220×4 = **880 kcal**
- 脂肪：60×9 = **540 kcal**
合计：**1980 kcal ≈ 2000 kcal**

这不是暴食，只是训练日的正常配比。

### 什么时候 220g 碳水合理？
- 力量训练 **3–5 次/周**
- 步数 **8k–12k/天**
- 目标是 **增肌/重组**（不是天天饿着快速掉秤）

### 什么时候可能不合适？（要下调）
- 活动量很小（长期 **3k–5k 步**）
- 体重较轻/食量小
- 血糖控制差（糖尿病/糖前期/空腹血糖偏高）
- 目标更偏“立刻瘦”

### 更靠谱的定法：训练日碳水按体重给范围
训练日实用起步：**2–3 g/kg 体重**
- 60kg：**120–180g**
- 70kg：**140–210g**
- 80kg：**160–240g**

所以：220g 对 70–80kg 且训练量足的人很正常；对 55–60kg 可能更像 150–180g。

### 更保守但仍能增肌的替代版（起步可用）
- **训练日**：**1900 kcal / P140 / C170 / F65**
- **非训练日**：**1750 kcal / P140 / C130 / F70**

## 6）要把数字一次性定准，只要 3 个信息
1) 体重（kg）  
2) 现在平均步数（大概就行）  
3) 血糖情况：有/没有/不知道（糖尿病/糖前期/空腹血糖偏高）
